What is BMI?

BMI (Body Mass Index) is a measure that uses height and weight to estimate body fat. It helps determine if a person is underweight, normal weight, overweight, or obese.



How to Calculate BMI?

BMI is calculated using the formula: BMI = weight (kg) / (height (m) * height (m)). A BMI value helps categorize individuals into different health risk groups.
